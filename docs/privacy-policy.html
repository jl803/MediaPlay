MediaPlay Privacy Policy

Last updated: April 3, 2026

MediaPlay is a local media player application for Android.

Information We Collect

MediaPlay does not collect, transmit, or share personal information with the developer or third parties.

Local Media Access

MediaPlay allows users to select media files stored on their device for local playback. Selected files are used only on the device to provide app functionality such as playback, playlists, and media browsing.

Local Storage

MediaPlay stores certain app data locally on the device, including:

  • playlists created by the user
  • media metadata needed for the library
  • generated video thumbnails
  • local app settings and preferences

This information is stored only on the user's device and is not sent to external servers.

Third-Party Services

MediaPlay does not use third-party analytics, advertising, or cloud-based user data collection services.

Children's Privacy

MediaPlay is not directed to children and is not designed for child-focused use.

Changes to This Policy

This privacy policy may be updated in the future. Any changes will be reflected on this page.
